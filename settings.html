  • For simplicity reasons, please do use min="0" max="100" low="0" high="50".
  • If the value is superior to 80, please do add setting-meterbar--high modifier and put high value to 80 (to display red color)
  • To sum up:
    • green between 0 and 50
    • orange between 50 and 80
    • Red above 80 (including values > 100), with setting-meterbar--high modifier class
  • Please do link meter tag to its description via aria-describedby.
